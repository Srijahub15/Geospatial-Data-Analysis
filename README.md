# Geospatial Data Analysis

## Objective

The objective of this project is to analyze location-based sales data to identify optimal regions for business expansion. The analysis examines sales performance across cities, states, and regions to discover high-demand markets and support strategic business decisions.

## Dataset

Sample Superstore Dataset

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

## Workflow

1. Imported required libraries.
2. Loaded the Superstore dataset.
3. Explored the dataset structure.
4. Checked for missing values.
5. Removed duplicate records.
6. Analyzed sales by region.
7. Visualized regional sales.
8. Identified top-performing states.
9. Analyzed city-wise sales.
10. Identified business expansion opportunities.
11. Performed correlation analysis.
12. Exported the cleaned dataset.

## Output Files

- geospatial_data_analysis.ipynb
- cleaned_superstore_sales.csv

## Conclusion

The analysis identified high-performing regions, states, and cities based on sales and profit. These insights can help businesses prioritize expansion into profitable markets while improving decision-making through location-based analysis.
